# Pomodoro Timer

Temporizador pomodoro construido con React.

## Funcionalidades
- Ciclos de 25 min foco / 5 min descanso
- Descanso largo de 15 min al finalizar las sesiones
- Sesiones configurables
- Reset del timer

## Tecnologías
- React
- useState, useEffect, useRef

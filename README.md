# Pomodoro Timer

Temporizador pomodoro construido con React.

## Funcionalidades
- Sesiones de 25 min foco / 5 min descanso
- Descanso largo de 15 min al finalizar las sesiones
- Configuracion de Sesiones(cuantas sesiones se quieren)
- Reset del timer por default a 1 sesion

## Tecnologías
- React
- useState, useEffect, useRef

Solidificando el uso del useState y useEffect, y utilizando nuevo hook useRef
